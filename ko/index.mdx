BIZ MORI API는 디지털 콘텐츠를 무단 사용으로부터 보호하기 위한 세 가지 핵심 서비스를 제공합니다.

## 서비스

<Columns cols={3}>
  <Card title="Anti-AI" icon="shield" href="/ko/api-reference/anti-ai/create-order">
    이미지가 AI 모델 학습에 사용되지 않도록 보호합니다. AI 시스템이 이미지를 학습하지 못하도록 보이지 않는 보호막을 적용합니다.
  </Card>
  <Card title="워터마크 삽입" icon="stamp" href="/ko/api-reference/watermark-embed/create-order">
    이미지에 보이지 않는 디지털 워터마크를 삽입합니다. 눈에 보이는 변형 없이 소유권을 추적하고 진위를 증명합니다.
  </Card>
  <Card title="워터마크 추출" icon="magnifying-glass" href="/ko/api-reference/watermark-extract/create-order">
    이미지에서 워터마크를 추출하고 검증합니다. 삽입된 워터마크 데이터를 감지하여 이미지 출처를 확인합니다.
  </Card>
</Columns>

## 기본 URL

모든 API 요청은 다음 주소로 보내야 합니다:

```
https://morimori.app/api/v2
```

## 응답 형식

**성공 응답:**
```json
{
  "data": { ... }
}
```

**오류 응답:**
```json
{
  "code": "ERROR_CODE"
}
```

HTTP 상태 코드가 오류 유형을 나타냅니다. 클라이언트는 i18n/로컬라이제이션에 오류 `code` 필드를 사용해야 합니다.
