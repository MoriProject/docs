**BIZ MORI API**는 디지털 콘텐츠 보호 서비스에 대한 프로그래매틱 접근을 제공합니다. 간단한 REST API를 통해 이미지를 무단 AI 학습으로부터 보호하고, 보이지 않는 소유권 워터마크를 삽입하며, 워터마크 진위를 검증하세요.

## 기본 URL

```
https://morimori.app/api/v2
```

모든 엔드포인트는 **HTTPS**로만 제공됩니다. HTTP 요청은 지원되지 않습니다.

## 요청 형식

`Content-Type: application/json`으로 요청을 보내고 `Authorization` 헤더에 API 키를 포함합니다:

```
Authorization: Bearer YOUR_API_TOKEN
```

## 응답 형식

모든 응답은 JSON을 반환합니다.

**성공**
```json
{
  "data": { ... }
}
```

**오류**
```json
{
  "code": "ERROR_CODE"
}
```

HTTP 상태 코드로 오류 범주를 파악하고, `code` 필드로 프로그래매틱 처리 및 로컬라이제이션을 수행하세요. 전체 목록은 [오류 코드](/ko/errors)를 참고하세요.
