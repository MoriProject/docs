## 기본 URL

```
https://morimori.app/api/v2
```

## 인증

모든 엔드포인트는 `X-API-Key` 헤더를 통한 API 키 인증이 필요합니다:

```
X-API-Key: sk_your_api_key_here
```

자세한 내용은 [인증](/ko/authentication) 가이드를 참고하세요.
