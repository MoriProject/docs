**최종 업데이트:** 2025년 2월 1일

MORI Inc.("MORI", "당사")는 귀하의 개인정보 보호를 위해 최선을 다합니다. 이 개인정보처리방침은 귀하가 BIZ MORI API 및 개발자 센터("서비스")를 이용할 때 당사가 정보를 어떻게 수집, 이용, 저장 및 공유하는지를 설명합니다.

---

## 1. 수집하는 정보

### 계정 정보
개발자 센터 회원가입 시 수집하는 정보:
- 이름 및 이메일 주소
- 조직명 (선택사항)
- 결제 정보 (결제 대행사를 통해 처리)

### API 이용 데이터
API 이용 시 자동으로 수집되는 정보:
- API 요청 로그 (엔드포인트, 타임스탬프, 응답 코드, 응답 시간)
- 주문 메타데이터 (주문 ID, 파일 수, 처리 상태)
- IP 주소 및 사용자 에이전트

### 콘텐츠 데이터
API에 제출된 파일 및 URL은 처리 목적으로만 일시적으로 사용됩니다. **요청된 처리 완료에 필요한 범위를 초과하여 원본 콘텐츠를 영구 저장하지 않습니다.**

---

## 2. 정보 이용 목적

수집된 정보는 다음 목적으로 이용됩니다:

- 서비스 제공 및 운영
- API 요청 인증 및 요청 제한 적용
- 남용, 사기 및 보안 사고 탐지 및 예방
- 익명화된 집계 이용 통계 생성
- 거래 관련 통신 (주문 상태, 결제) 발송
- 서비스 품질 개선 및 신기능 개발

---

## 3. 데이터 보유 기간

| 데이터 유형 | 보유 기간 |
|-----------|---------|
| 계정 정보 | 계정 유지 기간 + 삭제 후 30일 |
| API 요청 로그 | 90일 |
| 처리된 결과 파일 | 주문 완료 후 7일 |
| 결제 기록 | 관련 법령에서 요구하는 기간 (일반적으로 5년) |

---

## 4. 정보 제공

당사는 귀하의 개인정보를 **판매하지 않습니다**. 다음 경우에 한해 데이터를 공유할 수 있습니다:

- **인프라 제공업체** (클라우드 스토리지, CDN 등): 적절한 계약 하에 처리자로 운영
- **결제 처리업체**: 결제 목적
- **법 집행 기관 또는 규제 당국**: 법률에 따라 요구되거나 당사의 권리 보호를 위해 필요한 경우

---

## 5. 데이터 보안

당사는 다음과 같은 업계 표준 보안 조치를 시행합니다:

- 전송 중 모든 데이터에 TLS 암호화 적용
- 저장된 데이터에 저장 시 암호화 적용
- 권한이 있는 직원으로만 접근 제한
- 정기적인 보안 점검 실시

---

관할 지역에 따라 귀하는 다음 권리를 가질 수 있습니다:

| 권리 | 설명 |

## 7. 국제 데이터 이전

귀하의 데이터는 귀하의 거주 국가 외부에서 처리될 수 있습니다. 당사는 모든 국제 이전에 적절한 보호 장치(EU 데이터의 경우 표준 계약 조항 등)를 갖추고 있습니다.

---

## 8. 쿠키 및 추적

개발자 센터는 인증 및 세션 관리를 위한 필수 쿠키만 사용합니다. 제3자 광고 쿠키는 사용하지 않습니다.

---

## 9. 아동 개인정보 보호

서비스는 만 18세 미만의 개인을 대상으로 하지 않습니다. 당사는 미성년자의 개인정보를 고의로 수집하지 않습니다.
