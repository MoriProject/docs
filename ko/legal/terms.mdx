이 이용약관("약관")은 MORI Inc.("MORI", "당사")가 제공하는 BIZ MORI API 및 관련 개발자 서비스("서비스")에 대한 귀하의 접근 및 이용을 규율합니다. 서비스에 접근하거나 이용함으로써 귀하는 본 약관에 구속되는 것에 동의합니다.

---

## 1. 자격 요건 및 계정

- 귀하는 만 18세 이상이어야 하며 법적 구속력 있는 계약을 체결할 수 있는 자격이 있어야 합니다.
- 귀하는 API 키의 기밀 유지와 계정에서 발생하는 모든 활동에 대한 책임을 집니다.
- 계정에 무단 접근이 의심되는 경우 즉시 [mori@morimori.app](mailto:mori@morimori.app)으로 알려야 합니다.

## 2. 허용된 이용

귀하는 다음 목적으로만 서비스를 이용할 수 있습니다:

- 자체 애플리케이션 또는 워크플로에 BIZ MORI API 기능 통합
- 필요한 권리, 라이선스 또는 허가를 보유한 디지털 콘텐츠 처리
- 모든 관련 법령 및 본 약관을 준수하는 서비스 구축 및 운영

## 3. 금지된 이용

귀하는 서비스를 다음 목적으로 이용할 수 **없습니다**:

- 제3자의 지식재산권을 침해하는 콘텐츠 처리
- 서비스의 기반 알고리즘 역공학, 디컴파일 또는 추출 시도
- 사전 서면 동의 없이 제3자에게 원시 API 접근권 재판매 또는 재라이선스
- 서비스의 중단, 저하 또는 손상을 유발하는 활동
- 관련 법령 또는 규정 위반

## 4. API 키 및 보안

- API 키는 개인적이며 양도할 수 없습니다.
- API 키를 안전하게 보관해야 하며 공개적으로 접근 가능한 코드에 포함해서는 안 됩니다.
- 당사는 침해되었거나 본 약관을 위반하여 사용된 API 키를 취소할 권리를 보유합니다.

## 5. 서비스 수준 및 가용성

- 당사는 높은 가용성 유지를 위해 노력하지만 중단 없는 접근을 보장하지 않습니다.
- 계획된 유지보수는 가능한 경우 사전에 공지합니다.
- 요청 제한은 구독 플랜에 따라 적용됩니다. 제한을 초과하면 일시적 서비스 중지가 발생할 수 있습니다.

## 6. 지식재산권

- MORI는 서비스의 모든 권리, 소유권 및 이익(기반 기술, 모델 및 알고리즘 포함)을 보유합니다.
- 귀하는 제출하는 콘텐츠의 소유권을 보유합니다. 콘텐츠를 제출함으로써 귀하는 서비스 제공 목적으로만 해당 콘텐츠를 처리할 수 있는 제한적, 비독점적 라이선스를 MORI에 부여합니다.

## 7. 요금 및 결제

- 가격은 구독 시점에 개발자 센터에 게시된 내용을 기준으로 합니다.
- 관련 법령에 의해 달리 명시되지 않는 한 모든 요금은 환불되지 않습니다.
- 당사는 30일 사전 공지 후 가격을 변경할 권리를 보유합니다.

## 8. 해지

- 귀하는 언제든지 [mori@morimori.app](mailto:mori@morimori.app)으로 연락하여 계정을 해지할 수 있습니다.
- 귀하가 본 약관을 위반하는 경우, 위반의 심각성에 따라 사전 공지 여부와 관계없이 당사가 접근을 중지하거나 해지할 수 있습니다.
- 해지 시 귀하의 서비스 이용 권리는 즉시 소멸됩니다.
