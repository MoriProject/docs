BIZ MORI API provides three core services for protecting digital content from unauthorized use.

## Services

<Columns cols={3}>
  <Card title="Anti-AI" icon="shield" href="/api-reference/anti-ai/create-order">
    Protect images from being used for AI model training. Apply invisible protection that prevents AI systems from learning from your images.
  </Card>
  <Card title="Watermark Embed" icon="stamp" href="/api-reference/watermark-embed/create-order">
    Embed invisible digital watermarks into images. Track ownership and prove authenticity without visible alterations.
  </Card>
  <Card title="Watermark Extract" icon="magnifying-glass" href="/api-reference/watermark-extract/create-order">
    Extract and verify watermarks from images. Detect embedded watermark data to confirm image origin.

## Base URL

All API requests should be made to:

```
https://morimori.app/api/v2
```

## Response format

**Success response:**
```json
{
  "data": { ... }
}
```

**Error response:**
```json
{
  "code": "ERROR_CODE"
}
```

The HTTP status code conveys the error category. Clients should use the error `code` field for i18n/localization.
